# Space Shooter Game 🎮
A 2D arcade-style shooter game built with JavaScript and p5.js.

## Features
- Object-Oriented Programming (OOP) design with inheritance
- Classes for Player, Enemy, Bullet, and PowerUps
- Integrated images (spaceship, enemies, heart & shield icons)
- Sound effects (laser, explosion, background music)
- Game states: Start, Playing, Game Over
- PowerUps: Heart (extra life), Shield (temporary protection)

How to run: 
## How to Run
1. Clone or download this repository.
2. Make sure all files (`index.html`, `sketch.js`, `style.css`, and `assets/` folder) are in the same directory.
3. Open `index.html` in any modern browser (Chrome, Edge, Firefox).
4. Press **ENTER** to start the game.
5. Use **Arrow keys** to move the spaceship.
6. Press **Spacebar** to shoot bullets.
7. Collect PowerUps:
   - Heart → adds extra life
   - Shield → gives temporary protection
