# Spaceship Shooter Game

An arcade‑style shooter game built with **JavaScript** and **p5.js**.  
Fly your spaceship, shoot enemies, collect power‑ups, and survive as long as you can!  
Now playable on **PC and Mobile** (with on‑screen buttons).

---

##  Features
- Scrolling space background
- Player spaceship with movement controls
- Enemy ships with increasing speed
- Bullets with shooting sound effects
- Power‑ups: ❤️ extra life and star shield
- Background music and explosion effects
- Score and lives tracking
- Game over and restart screen
- **Mobile support with on‑screen buttons (move & fire)**

---

##  Controls

###  PC / Laptop
- **Arrow Keys** → Move spaceship (Up, Down, Left, Right)  
- **Spacebar** → Fire bullets  
- **Enter** → Start / Restart game  

###  Mobile / Android
- On‑screen buttons for movement: ⬅️ ➡️ ⬆️ ⬇️  
- 🔥 FIRE button to shoot bullets  
- Tap **Start** button (or Enter key if keyboard available) to begin game  

---

##  How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/anshitatomar04122004-oss/space-shooter-game.git
2. Open index.html in any modern browser (Chrome, Edge, Firefox).
3. Press ENTER to start the game.
4. Use Arrow keys to move the spaceship.
5. Press Spacebar to shoot bullets.
